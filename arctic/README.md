# Arctic Theme ❄️

A refreshing, cool-toned theme inspired by pristine ice and snow.

## Features

- Clean, professional design with cool blues
- Responsive layout for all devices
- Component showcase: buttons, forms, cards, tables
- Accessible and semantic HTML

## Quick Start

1. Open `index.html` in your browser
2. Customize colors in `css/styles.css`
3. Modify content to fit your needs

## Files

- `index.html` — Demo page with all components
- `css/styles.css` — All styling
- `js/main.js` — Interactive components (modal, tabs)

## Color Palette

- Primary: `#0ea5e9` (Sky Blue)
- Secondary: `#7dd3fc` (Light Blue)  
- Background: `#0c19.29` (Dark Navy)
