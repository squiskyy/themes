# Atlantis Theme 🌊

Deep ocean-inspired theme with glass morphism effects.

## Features

- Glass morphism card design
- Chart.js integration for data visualization
- Admin dashboard layout
- Responsive navigation

## Quick Start

1. Open `index.html` in your browser
2. Customize colors in `css/colors.css`
3. Modify content to fit your needs

## Files

- `index.html` — Dashboard demo
- `css/styles.css` — All styling
- `css/colors.css` — Color variables
- `js/main.js` — Chart.js and interactions

## Color Palette

- Primary: `#06b6d4` (Cyan)
- Secondary: `#0891b2` (Teal)
- Background: `#0c1929` (Dark Navy)
