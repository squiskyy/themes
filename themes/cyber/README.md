# Cyber Theme 💾

Neon-futuristic theme with glowing cyberpunk aesthetics.

## Features
- Neon glow effects
- Dark terminal-inspired design
- Matrix-style green accents
- Tech-forward aesthetic

## Quick Start
1. Open `index.html` in your browser
2. Customize colors in `css/colors.css`
3. Modify content

## Color Palette
- Primary: `#00ff9d` (Neon Green)
- Secondary: `#00ffff` (Cyan)
- Background: `#0.00.14` (Near Black)
