# Ember Theme 🔥

Bold fiery theme with passionate orange and red tones.

## Features
- Warm gradient backgrounds
- Energetic, bold design
- High-contrast accents
- Attention-grabbing aesthetic

## Quick Start
1. Open `index.html` in your browser
2. Customize colors in `css/colors.css`
3. Modify content

## Color Palette
- Primary: `#f97316` (Orange)
- Secondary: `#fbbf24` (Yellow)
- Background: `#1c0400` (Dark Maroon)
